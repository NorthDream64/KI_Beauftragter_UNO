# Workflow für Quellennutzung:
#
# Priorität 1: project_knowledge_search → relevante PDF-Inhalte direkt lesen
# Priorität 2: web_search mit Schlagwörtern aus "Schlagwörter"-Spalte
# Priorität 3: User gibt URL direkt im Chat ein → web_fetch
#
# Halluzinations-Schutz:
# Seitenangaben aus Projektdokumenten immer direkt aus PDF verifizieren,
# nicht aus dem Gedächtnis nennen. Kursnotizen (KIBeauftragter_Ulrichs_
# Kursnotizen.pdf) inhaltlich verwenden, aber NICHT als Quelle zitieren.
#
# AISI-Quellen: Besonders geeignet für Sicherheitsthemen (Prompt Injection,
# Agenten-Sicherheit, Frontier AI) — staatliche, peer-reviewed Forschung.
#
# Beispiel:
# User: "Was sagt Rohrlich zur Haftung des KI-Beauftragten?"

# → Direktzitat mit Seitenangabe aus KI_und_Recht.pdf
